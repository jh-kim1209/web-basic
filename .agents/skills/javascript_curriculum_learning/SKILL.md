# 자바스크립트 주차별 미션 학습 안내 스킬

이 스킬은 사용자가 자바스크립트 학습 과정에서 개념을 확실히 잡고 문법을 배울 수 있도록 돕는 동작 지침을 규정합니다.

## 핵심 행동 지침

1. **자동 구현 배제 및 설명 우선**:
   * 새로운 주차별 미션 디렉토리(예: `week-02`)가 지정되었을 때, 코드를 자동으로 파일에 바로 삽입하거나 완성형 기획서를 생성하지 마십시오.
   * 사용자의 명시적인 구현 요청이 있기 전까지는 원리 분석과 문법 해설을 채팅을 통해 먼저 설명하십시오.

2. **개념 설명서 (`explanation.md`) 분리 작성**:
   * 미션 코드의 동작 원리(예: 클로저, 비동기 처리, 이벤트 루프 등)를 상세히 설명하는 `explanation.md` 파일을 해당 주차 폴더에 생성하십시오.
   * 단순히 코드만 적는 것이 아니라 사용자가 자바스크립트 문법(Spread, Array Methods, Object Keys 등)을 이해할 수 있도록 문법적 해설을 단계별로 제공해야 합니다.

3. **`README.md` 업데이트 및 연동**:
   * 주차별 `README.md` 하단에 `## 💡 학습 및 구현 참고 가이드` 섹션을 새로 추가하십시오.
   * 해당 주차에서 반드시 지켜야 할 핵심 규칙(예: 불변성 유지, 최적화 등)을 요약하고 `explanation.md` 링크를 연동하여 제공하십시오.

4. **자바스크립트 초급자 맞춤형 코드 및 문법 해설**:
   * 자바스크립트 문법을 잘 모르는 학습자를 배려하여, 사용된 고급/모던 문법(화살표 함수, 구조 분해 할당, 비동기 문법 등)에 대해 직관적인 비유와 코드 라인 단위 해설을 제공하십시오.
