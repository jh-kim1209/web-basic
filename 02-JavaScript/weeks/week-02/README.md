# 2주차: ES6+ 표준 문법 및 함수형 프로그래밍 입문

## 📖 핵심 요약
구조 분해 할당(Destructuring) 및 전개 구문(Spread Operator) 등 모던 ES6+ 자바스크립트의 필수 편의 문법을 완전히 익히고, 코드의 가독성과 부작용(Side Effect)을 줄이기 위한 **함수형 프로그래밍(Functional Programming)** 방법론을 도입합니다.

*   **일급 객체 (First-Class Object)**: 자바스크립트의 함수는 변수에 할당할 수 있고, 함수의 인자로 전달될 수 있으며, 다른 함수의 결과로 반환될 수 있는 일급 객체입니다.
*   **고차 함수 (Higher-Order Function)**: 함수를 인자로 받거나 함수를 반환하는 함수로, 대표적으로 `map`, `filter`, `reduce` 등이 있습니다.
*   **커링 (Currying)**: 다중 인자를 받는 함수를 단일 인자를 받는 일련의 함수들의 연쇄로 변환하는 기법입니다. 부분 적용(Partial Application)을 가능하게 합니다.
*   **함수 합성 (Composition)**: 하나의 함수의 출력이 다른 함수의 입력이 되도록 함수들을 연결하여 크고 복잡한 연산을 잘게 쪼갠 함수들의 결합으로 해결하는 기법입니다.

*   **미션 파일**: `mission.js`
*   **요구 사항**:
    1.  `pipe` 함수는 여러 개의 단일 인자 함수들을 입력받아, 입력받은 순서대로 실행하고 최종 결과를 반환하는 합성 함수를 만들어내야 합니다.
    2.  `curry` 함수는 임의의 개수를 지닌 인자들을 입력받는 다중 인자 함수를 커링 가능한 함수로 변환해 주어야 합니다.
    3.  커링 함수와 고차 함수를 조합하여 아래의 요구 사항을 처리하는 선언형 데이터 정제 함수 `processUsers(users)`를 작성하십시오.
        -   나이가 20세 이상인 유저만 필터링합니다. (`filter`)
        -   필터링된 유저의 정보 중 이름과 이메일 주소만 포함하는 객체로 맵핑합니다. (`map`)
