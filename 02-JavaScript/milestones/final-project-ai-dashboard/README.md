# 최종 프로젝트: 실시간 AI 연동 협업 대시보드 (7~8주차)

## 📌 목표
전체 8주 과정을 마무리하며, 1~6주간 개발해온 핵심 코어 기능(상태 관리 스토어, TypeScript 정제 로직, Web Workers 병렬 처리) 위에 **React/Next.js** 프레임워크와 **Zustand 전역 스토어**를 올리고, **AI LLM API**를 스트리밍(SSE)으로 연동하여 실무 수준의 종합 풀스택 협업 대시보드 애플리케이션을 최종 완성하고 배포합니다.

---

## 🛠️ 요구 사항
1.  **React/Zustand 컴포넌트 마운트**:
    *   커스텀 대시보드 UI를 React 컴포넌트로 승화시킵니다.
    *   글로벌 상태 관리를 위해 Zustand 스토어를 도입하고, 사용자 검색 필터 및 실시간 렌더링에 동기화합니다.
2.  **실시간 AI 스트리밍 연동**:
    *   대시보드 내에 AI 어시스턴트 채팅 영역 또는 보고서 생성기 기능을 탑재합니다.
    *   사용자의 특정 요청 프롬프트를 전송하면, AI 서버가 반환하는 스트리밍 바디(ReadableStream)를 읽어 `onChunk` 콜백 함수로 실시간 화면에 글자 단위로 뿌려주는 단방향 이벤트 스트리밍(SSE) UI를 완성합니다.
3.  **프로덕션 배포 파이프라인**:
    *   완성된 웹 서비스를 컨테이너 형태로 패키징하는 `Dockerfile` 및 `.dockerignore`를 설계합니다.
    *   Vercel 또는 대체 클라우드 호스팅 서비스를 연계하여 실제 인터넷 주소 상에서 누구나 접근할 수 있도록 최종 배포 처리를 마칩니다.
